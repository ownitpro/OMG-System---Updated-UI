## Accessibility Standards

### WCAG 2.1 AA Compliance

All color combinations meet WCAG 2.1 Level AA standards for contrast ratios:
- Normal text: minimum 4.5:1
- Large text (18pt+): minimum 3:1
- UI components: minimum 3:1

### Verified Contrast Ratios

| Foreground | Background | Ratio | Pass |
|------------|------------|-------|------|
| Slate 900 | White | 19.1:1 | AAA |
| Slate 500 | White | 4.7:1 | AA |
| Indigo 600 | White | 5.2:1 | AA |
| White | Indigo 600 | 5.2:1 | AA |
| Slate 50 | Slate 900 | 18.2:1 | AAA |
| Green 600 | White | 4.1:1 | AA (Large) |
| Red 600 | White | 5.7:1 | AA |

## Color Naming Convention

When implementing in code, use the following naming convention:

```css
/* CSS Custom Properties */
--color-primary: #4F46E5;
--color-primary-hover: #6366F1;
--color-primary-active: #4338CA;

--color-background: #F8FAFC;
--color-surface: #FFFFFF;
--color-border: #CBD5E1;

--color-text-primary: #0F172A;
--color-text-secondary: #64748B;
--color-text-disabled: #94A3B8;

--color-success: #16A34A;
--color-warning: #D97706;
--color-error: #DC2626;
--color-info: #2563EB;
```

```js
// JavaScript/TypeScript
const colors = {
  primary: '#4F46E5',
  secondary: '#64748B',
  success: '#16A34A',
  warning: '#D97706',
  error: '#DC2626',
  info: '#2563EB'
}
```

| Red 600 | C:0 M:83 Y:83 K:14 | Pantone 1795 C |

### Print Guidelines
- For offset printing, use Pantone spot colors when possible
- For digital printing, use CMYK values provided
- Minimum logo size: 0.75 inches (19mm) wide
- Maintain clear space equal to the height of the vault icon around all logos

---

## Brand Color Don'ts

- Do not use gradients with primary brand colors
- Do not tint or adjust opacity of primary brand colors below 100%
- Do not use colors outside this palette for brand materials
- Do not place light text on light backgrounds or dark text on dark backgrounds
- Do not use pure black (#000000) for text; use Slate 900 instead
