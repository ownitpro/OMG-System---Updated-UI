# ICPs & Use Cases

## Ideal Customer Profiles

### Business ICP

**Profile:** Small–mid firms (5–100 seats) managing recurring client paperwork

**Industries:**
- Accounting/bookkeeping firms
- Real estate brokerages
- Contractors and construction
- Project Management Offices (PMOs)
- Legal operations teams

**Jobs-to-be-done:**
- Intake client documents efficiently
- Request documents from clients
- Version control and document history
- Secure sharing with watermarks
- Audit trails for compliance
- Basic analytics and reporting
- Cost visibility and control

**Pain Points:**
- Chasing files via email
- Security concerns with file sharing
- Lack of audit trails
- Hidden costs and bill shock
- Client confusion with complex tools

### Personal ICP

**Profile:** Busy professionals & families consolidating documents

**Use Cases:**
- Tax preparation document organization
- Immigration document packs
- Vehicle/home records management
- Receipt and bill organization
- Sharing documents with accountants
- Personal document vault

**Pain Points:**
- Documents scattered across devices
- Hard to find important files
- Sharing sensitive documents securely
- Organizing receipts and bills
- Preparing documents for tax season

## Top Use Cases

### Business Use Cases

1. **New-client onboarding packs**
   - Create portal for new client
   - Send document request template
   - Client uploads all required docs
   - Everything organized automatically

2. **Monthly receipt/statement intake**
   - Set up recurring request
   - Client uploads monthly statements
   - Auto-labeled and organized
   - Ready for review

3. **Deal/engagement rooms with expiry + watermark**
   - Create secure share link
   - Set PIN and expiry date
   - Watermark all documents
   - Track access and revoke when done

4. **Year-end export to accountant (bundle/zip)**
   - Organize all tax documents
   - Export as bundle
   - Share with accountant
   - Maintain audit trail

### Personal Use Cases

1. **Tax preparation**
   - Organize all tax documents
   - Label by tax year
   - Share with accountant
   - Maintain records

2. **Receipt management**
   - Forward receipts via Email-to-Vault
   - Auto-labeled and organized
   - Search by merchant or amount
   - Export for expense reports

3. **Important document storage**
   - IDs, passports, licenses
   - Insurance documents
   - Warranties and receipts
   - All in one secure place

4. **Family document sharing**
   - Share documents with family members
   - Time-boxed access
   - PIN protection
   - Revoke when needed

## Use Case Templates

### Template: Client Onboarding

**Title:** "Onboard New Clients in Minutes"

**Description:** Create a secure portal, send a document request, and get everything organized automatically.

**Steps:**
1. Create client portal
2. Send document request template
3. Client uploads documents
4. Documents auto-organized and labeled

### Template: Secure Sharing

**Title:** "Share Documents Without Risk"

**Description:** Create PIN-protected, time-boxed share links with watermarks.

**Steps:**
1. Select documents to share
2. Set PIN and expiry date
3. Generate secure link
4. Track access and revoke anytime

### Template: Document Intake

**Title:** "Stop Chasing Files"

**Description:** Request documents from clients with one secure link.

**Steps:**
1. Create document request
2. Choose template or custom
3. Send to client portal
4. Receive organized documents
