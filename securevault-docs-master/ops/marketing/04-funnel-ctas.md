# Funnel & CTAs

## Primary CTAs

**Get started free** → Signup → Onboarding wizard

## Secondary CTAs

- **Try the demo** → Demo page
- **Contact Sales** → Sales form

## Tertiary CTAs

- See pricing
- See features
- Explore marketplace

## CTA Hierarchy

### Homepage
1. **Primary:** Get started free (hero)
2. **Secondary:** Try the demo (hero)
3. **Tertiary:** See pricing, See features (below fold)

### Features Page
1. **Primary:** Try [Feature] Demo (per section)
2. **Secondary:** Get started free (footer)
3. **Tertiary:** See pricing, Contact Sales

### Pricing Page
1. **Primary:** Start free trial (per plan)
2. **Secondary:** Contact Sales (Enterprise)
3. **Tertiary:** See FAQs, Try demo

### Demo Page
1. **Primary:** Get started free (after demo)
2. **Secondary:** Contact Sales
3. **Tertiary:** See pricing

## Deflection Logic

### Pricing Deep Link

**If user clicks pricing from inside an org:**
- Deep-link with current plan preselected
- Show upgrade path if applicable
- Highlight current usage vs. plan limits

### Demo Routing

**Demo page captures intent:**
- Industry (multi-select)
- Role (dropdown)
- Team size (dropdown)
- Route to tailored demo (Business/Personal)

**Routing Logic:**
- Business use case → Business Demo
- Personal use case → Personal Demo
- Undecided → Interactive Walkthrough

## Conversion Paths

### Path 1: Direct Signup
Homepage → Get started free → Signup → Onboarding

### Path 2: Demo First
Homepage → Try the demo → Demo page → Get started free → Signup

### Path 3: Research First
Homepage → Features → Pricing → Get started free → Signup

### Path 4: Sales Assisted
Homepage → Contact Sales → Sales form → Demo call → Signup

## A/B Testing Ideas

### CTA Copy
- "Get started free" vs. "Start free trial"
- "Try the demo" vs. "See it in action"
- "Contact Sales" vs. "Talk to an expert"

### CTA Color
- Green (primary) vs. Blue
- High contrast vs. Subtle

### CTA Size
- Large vs. Medium
- Full width vs. Inline
