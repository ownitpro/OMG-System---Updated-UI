# Demo Scripts

## Customer-Facing Demo Guides

Use these scripts for sales demos and customer presentations.

## Business Demo (7-9 minutes)

### 1. Hero: Overview KPIs and Quick Actions (1 min)

**What to Show:**
- Overview dashboard
- KPI strip (workspace usage, portals, requests, approvals)
- Quick Actions panel
- Getting Started checklist

**What to Say:**
> "Welcome to SecureVault Docs Business. Here's your overview dashboard showing key metrics at a glance. You can see workspace usage, active client portals, open requests, and pending approvals. Quick Actions let you jump to common tasks instantly."

### 2. Upload + Label; Instant Search (1.5 min)

**What to Show:**
- Go to Documents
- Upload a file (drag & drop or file picker)
- Show progress bar
- Apply label (e.g., "Tax Document")
- Use search bar to find file by label

**What to Say:**
> "Uploading documents is simple - just drag and drop or use the file picker. Files are automatically processed with OCR. You can apply labels for quick organization. Search is instant - find any document by name, label, or content."

### 3. Create Client Portal; Send Request (2 min)

**What to Show:**
- Go to Client Portals
- Click "Create Portal"
- Enter client name and email
- Send invite (mock)
- Go to Requests
- Create new request from template
- Select portal and set due date
- Send request

**What to Say:**
> "Client portals give each client their own secure space. Create a portal, invite your client, and they get secure access. Then send document requests using templates - like KYC documents or year-end packages. The client uploads directly into their portal, and you see everything organized."

### 4. Share with PIN/Expiry; Watermark Viewer (1.5 min)

**What to Show:**
- Go to Shares
- Create new share link
- Select document
- Set PIN (e.g., "1234")
- Set expiry (e.g., 7 days)
- Generate link
- Open link in incognito window
- Enter PIN
- Show watermark viewer

**What to Say:**
> "Sharing is secure and controlled. Create a share link with optional PIN and expiry date. Recipients see a watermarked viewer - no downloads unless you allow it. You can revoke access anytime, even if the link was already shared."

### 5. Analytics Trends; Billing Caps; Upgrade Flow (1.5 min)

**What to Show:**
- Go to Analytics
- Show charts (uploads/day, OCR pages, trends)
- Go to Billing
- Show usage bars
- Point out alert thresholds
- Click "Upgrade" button
- Show plan comparison

**What to Say:**
> "Analytics show your usage trends and help you understand activity. Billing is transparent - you see exactly where you are against your plan limits. Alerts warn you at 70, 80, 90, and 95 percent. Upgrading is one click away."

### 6. Install App (PWA/Desktop) (0.5 min)

**What to Show:**
- Go to Settings → Install App
- Show PWA instructions
- Show desktop installer options
- Note: PWA and desktop only (no mobile apps)

**What to Say:**
> "Install SecureVault Docs as a PWA on any device, or download the desktop app for Windows, Mac, or Linux. Works offline and syncs when you're back online."

## Personal Demo (5-7 minutes)

### 1. Vault Folders + Labels; Receipts Example (2 min)

**What to Show:**
- Go to Vault
- Show folder structure (IDs, Bills, Receipts, Tax)
- Show monthly subfolders (2025/January, etc.)
- Upload a receipt
- Apply labels (Receipt, Home)
- Show how labels drive search

**What to Say:**
> "Your personal vault is organized with folders and labels. Upload receipts, bills, tax documents - anything. Labels make search instant. Monthly subfolders help you organize by time period."

### 2. Share Link to Accountant; Revoke (1.5 min)

**What to Show:**
- Go to Shares
- Create new share link
- Select tax documents
- Set PIN
- Set expiry
- Generate link
- Show how to share with accountant
- Then revoke the link
- Show link no longer works

**What to Say:**
> "Share documents securely with your accountant or anyone else. Set a PIN and expiry date. The recipient sees a watermarked viewer. When tax season is over, revoke the link with one click."

### 3. Email-to-Vault Mock; File Lands Labeled (1 min)

**What to Show:**
- Go to Settings → Integrations
- Show Email-to-Vault address
- Explain forwarding receipts
- Show how files auto-land in vault
- Show auto-labeling (mocked in test)

**What to Say:**
> "Forward receipts and bills to your unique Email-to-Vault address. Files automatically land in your vault and get labeled. Perfect for capturing receipts on the go."

### 4. Simple Analytics; Trial Reminder (0.5 min)

**What to Show:**
- Go to Analytics
- Show simple charts (uploads/month, top labels)
- Go to Overview
- Point out trial reminder banner

**What to Say:**
> "Analytics show your usage patterns. See what you're uploading most, which labels you use. The trial reminder keeps you informed about your plan status."

## Demo Tips

### Preparation

1. **Test First**
   - Run through demo script before customer call
   - Verify all features work
   - Check demo org data is seeded

2. **Know the Flow**
   - Memorize key talking points
   - Practice transitions
   - Prepare for questions

3. **Have Backup**
   - Screenshots ready
   - Alternative demo org
   - Troubleshooting steps

### During Demo

1. **Engage Customer**
   - Ask about their use case
   - Tailor demo to their needs
   - Pause for questions

2. **Highlight Benefits**
   - Focus on value, not features
   - Show time savings
   - Demonstrate security

3. **Handle Questions**
   - Refer to FAQ (see `14-faq.md`)
   - Be honest about limitations
   - Note test mode vs production

### After Demo

1. **Follow Up**
   - Send Contact Sales form
   - Provide next steps
   - Schedule follow-up call

2. **Document**
   - Note customer questions
   - Record feature requests
   - Update demo script if needed
