# Onboarding Checklists

## Day 1 Checklists by Role

### Admin (Business)

**Goal:** Set up an organization and complete first workflows.

1. **Create org**
   - Go to Settings → Organization
   - Set organization name, logo, and region
   - Save changes

2. **Invite team members**
   - Go to Settings → Team & Roles
   - Invite members by email
   - Assign roles (Admin or Member)
   - Verify invite emails (mock in test)

3. **Create first Client Portal**
   - Go to Client Portals
   - Click "Create Portal"
   - Enter client name and email
   - Send invite (mock email in test)

4. **Send first Request**
   - Go to Requests
   - Click "New Request"
   - Choose a template (or create custom)
   - Select portal and set due date
   - Send request

5. **Test Share with PIN/expiry**
   - Go to Shares
   - Create new share link
   - Set PIN and expiry (e.g., 7 days)
   - Copy link and test in incognito window
   - Verify watermark viewer

6. **Review Billing caps**
   - Go to Billing
   - Review usage bars (mock values in test)
   - Check alert thresholds (70/80/90/95%)
   - Understand upgrade path

### Personal User

**Goal:** Set up personal vault and test sharing.

1. **Upload and organize**
   - Go to Vault
   - Upload 3 files (e.g., ID, bill, receipt)
   - Apply labels (ID, Bills, Receipts)
   - Create folders if needed

2. **Create share link**
   - Go to Shares
   - Create new share link
   - Set PIN and expiry
   - Open link in incognito to verify
   - Test watermark viewer

3. **Try Install App**
   - Go to Settings → Install App
   - Follow PWA instructions (Add to Home Screen)
   - Or download desktop installer (mock in test)

### Support

**Goal:** Learn support tools and common workflows.

1. **Audit view**
   - Access audit logs (shared table)
   - Filter by user and action type
   - Review recent activity

2. **Account recovery**
   - Practice account recovery flow
   - Test invite re-send functionality
   - Verify email delivery (mock in test)

3. **Share troubleshooting**
   - Test expired share links
   - Verify PIN protection
   - Check watermark viewer accessibility

### Sales

**Goal:** Learn demo flows and lead routing.

1. **Run Business Demo**
   - Follow demo script (see `13-demo-scripts.md`)
   - Practice 7-9 minute walkthrough
   - Highlight key features

2. **Run Personal Demo**
   - Follow demo script
   - Practice 5-7 minute walkthrough
   - Focus on simplicity

3. **Contact Sales form**
   - Test form submission
   - Verify n8n webhook (see `ops/n8n/`)
   - Practice follow-up template

## Week 1 Goals

**Admin:**
- Complete onboarding checklist
- Set up 2-3 client portals
- Send 5+ document requests
- Create 3+ share links
- Review analytics

**Personal User:**
- Organize 10+ documents
- Create 2+ share links
- Install PWA or desktop app
- Set up Email-to-Vault

**Support:**
- Review all runbooks (see `11-support-runbooks.md`)
- Practice troubleshooting common issues
- Learn audit log navigation

**Sales:**
- Master both demo scripts
- Practice answering FAQ (see `14-faq.md`)
- Test Contact Sales form routing
